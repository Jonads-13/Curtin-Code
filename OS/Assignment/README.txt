README for 2023 Semester 1 Operating Systems Assignment 

Running "make" from the terminal in the directory containing the source code files will compile and create the executable.

"make clean" will delete all object files

To execute the program, run:

./cq m c w d i 

where:

m = Max length of the customer queue as a positive integer
c = Amount of time between each customer joins the queue as a positive integer
w = Amount of time to perform a withdrawal service as a positive integer
d = Amount of time to perform a deposit service as a positive integer
i = Amount of time to perform an information service as a positive integer
